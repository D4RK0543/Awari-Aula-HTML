Um pouco sobre CSS Grid Layout

O CSS Grid Layout é uma tecnologia do CSS que desde 2017 está presente na maioria dos principais navegadores modernos. Ela nos permite criar layouts baseados em grids de uma forma extremamente rápida, fácil e… padronizada. Apresentarei rapidamente nesse artigo um pouco das principais coisas que você pode fazer com essa feature.
